Build a glamorous, girlish, pink-themed personal portfolio website for **Nirdeshika Pandey** — a 21-year-old Computer & Data Science student, freelance model, brand ambassador, and driven dev girl from Kathmandu, Nepal. The design should feel like a luxury fashion magazine meets a tech portfolio — confident, feminine, bold, and unforgettable.

---

## 🎨 DESIGN DIRECTION

**Aesthetic:** Glamcore / Barbie-tech / Luxe Girly — think Vogue meets GitHub. NOT cute-pastel-soft. Think BOLD pink with gold accents, dramatic typography, editorial layouts.

**Color Palette (use CSS variables / Tailwind config):**
- Primary: Deep rose pink `#E91E8C` or hot pink `#FF2D78`
- Secondary: Blush `#FFB6C1`
- Accent: Champagne gold `#D4AF37`
- Dark base: Near-black `#0D0009` (deep plum-black)
- Surface cards: `#1A0010` or `rgba(255,45,120,0.07)`
- Text: White `#FFFFFF` and soft cream `#FFF0F5`

**Typography:**
- Display/Headings: Google Font `"Cormorant Garamond"` (bold, italic, editorial) or `"Playfair Display"`
- Body: `"DM Sans"` or `"Outfit"`
- Accent labels: `"Bebas Neue"` for tags and section headers

**Visual Details:**
- Animated gradient mesh background (pink-to-dark-plum) with a subtle grain/noise overlay
- Floating glitter/sparkle particles (CSS or simple JS canvas)
- Gold shimmer dividers between sections
- Glassmorphism cards with pink-tinted blur backdrop
- Custom cursor: a small glowing pink dot
- Smooth scroll with scroll-reveal fade-ins for each section
- Hover: cards lift with a pink glow box-shadow

## ⚙️ TECHNICAL REQUIREMENTS

- **Framework:** React (functional components + hooks)
- **Styling:** Tailwind CSS (extend config for custom colors/fonts)
- **Fonts:** Load via Google Fonts (`@import` in index.css)
- **Animations:** Tailwind `animate-*` + custom keyframes in tailwind.config for:
  - `shimmer` (gold shimmer sweep)
  - `float` (gentle up-down float)
  - `fadeInUp` (scroll reveal)
  - `typing` (typewriter cursor blink)
- **Particles:** Lightweight vanilla JS canvas sparkles OR CSS pseudo-element stars
- **Scroll:** `IntersectionObserver` for reveal-on-scroll effects
- **Responsive:** Fully mobile-first responsive (hamburger nav on mobile)
- **Cursor:** Custom CSS cursor with pink glow `mix-blend-mode: difference`
